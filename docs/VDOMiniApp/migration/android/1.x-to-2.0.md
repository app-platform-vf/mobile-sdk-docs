# Migration Guide: VDOMiniApp Android 1.x → 2.0

## Breaking Changes

### 1. SdkConfig — fields mới bắt buộc

v2.0 thêm `partnerKey`, `secretId`, và `variant` vào `SdkConfig`. Cả ba được cung cấp khi đăng ký trên hệ thống.

```kotlin
// v1.x
val sdkConfig = SdkConfig(
    apiKey = "..."
)

// v2.0
val sdkConfig = SdkConfig(
    apiKey = "...",
    partnerKey = "<partner-key>",
    secretId = "<secret-id>",
    variant = BuildVariant.Uat,
    hostAppApplicationName = getString(R.string.app_name),
    isLoadFromAsset = false
)
```

### 2. HostAppBridge — cách khởi tạo thay đổi

Từ v2.0, implement `HostAppBridge` trực tiếp bằng cách tạo một class:

```kotlin
class MyHostAppBridge : HostAppBridge {
    override fun getUserData(
        miniAppKey: String,
        dataName: List<String>,
        resultHandler: (Map<String, Any?>) -> Unit
    ) {
        val result = mutableMapOf<String, Any?>()
        // populate result...
        resultHandler(result)
    }
    override fun expiredSession(miniAppKey: String, data: Map<String, Any>) {}
    override fun observeLifecycle(miniAppKey: String, miniAppLifecycle: MiniAppLifecycle) {}
    override fun paymentStart(miniAppKey: String, request: String) {}
    override fun paymentEnd(miniAppKey: String, request: String) {}
    override fun paymentRequest(miniAppKey: String, data: Map<String, Any>) {}
    override fun intercept(miniAppKey: String, message: String) {}
}

MiniAppSdk.getInstance().hostAppBridge = MyHostAppBridge()
```

### 3. openMiniApp — thêm tham số `parameter`

`openMiniApp()` nhận thêm tham số tùy chọn `parameter: Map<String, List<String>>` để truyền query param vào URL của mini app:

```kotlin
MiniAppSdk.getInstance().openMiniApp(
    miniAppKey = miniAppKey,
    initInfo = initRequest,
    hostView = hostView,
    parameter = mapOf("param1" to listOf("value1"))
)
```

## APIs mới

| Method | Mô tả |
| ------ | ----- |
| `openMiniAppFromDeepLink(initInfo, hostView, qrContent, params)` | Mở mini app từ Adjust/QR link |
| `openMiniAppActivity(miniAppKey, initInfo, hostView, params)` | Mở mini app trong Activity mới |
| `HostAppBridge.observeLifecycle` | Theo dõi lifecycle của mini app |
| `HostAppBridge.paymentStart/End/Request` | Hooks cho luồng thanh toán |
| `HostAppBridge.intercept` | Intercept JS bridge message |
| `MiniAppThemeConfig` | Cấu hình toolbar (title, màu nền, màu chữ, ẩn/hiện) |
