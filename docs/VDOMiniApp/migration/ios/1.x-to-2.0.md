# Migration Guide: 1.x → 2.0 (iOS)

Hướng dẫn nâng cấp Mini App SDK từ version 1.x lên 2.0.

## Tổng quan thay đổi

| Mục                  | 1.x                              | 2.0                              |
| --------------------- | -------------------------------- | -------------------------------- |
| Distribution          | XCFramework thủ công             | Swift Package (VDOFrameworks)    |
| Version management    | Copy thủ công                    | Package.swift + checksum         |
| Minimum iOS           | 12.0                             | 12.0                             |

## Bước 1: Xoá xcframework references cũ

Trong Xcode:

1. Chọn target → **Build Phases → Link Binary With Libraries**
2. Xoá các `.xcframework` references cũ (VDOMiniApp, VDONetwork, VDOUtils)
3. Xoá group `xcframeworks/` khỏi Project Navigator (chỉ Remove Reference)

## Bước 2: Xoá Build Settings cũ

Trong **Build Settings**, xoá path tới xcframeworks nếu có:

```
FRAMEWORK_SEARCH_PATHS → Xoá path trỏ tới thư mục xcframeworks cũ
```

## Bước 3: Cài đặt SDK mới

Làm theo hướng dẫn tại [iOS Installation](../../integration/ios/installation.md):

1. Download `VDOFrameworks-2.1.0.zip`
2. Giải nén và đặt `VDOFrameworks/` vào root project
3. Add package vào Xcode

## Bước 4: Verify

```swift
import VDOMiniApp

// Build: ⌘B
```

## Lưu ý

- Cấu trúc project thay đổi: thư mục `xcframeworks/` được thay bằng `VDOFrameworks/`
- Các API sử dụng trong code **không thay đổi** — chỉ thay đổi cách distribution
- Xem [Release Notes 2.1.0](../../releases/ios/2.1.0.md) để biết thêm chi tiết
