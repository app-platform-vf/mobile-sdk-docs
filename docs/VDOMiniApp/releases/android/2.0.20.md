# Release Notes - VDOMiniApp Android 2.0.20

**Version:** 2.0.20
**Platform:** Android

## Changelog

### Added
- `openMiniAppFromLinkQR()` — open mini app from Adjust link, Adjust QR, or QR data link
- `SdkConfig.partnerKey` and `SdkConfig.secretId` — partner authentication credentials
- `HostAppListener.observeLifecycle()` — observe mini app lifecycle state changes
- `HostAppListener.paymentStart()`, `paymentEnd()`, `paymentRequest()` — payment flow hooks
- `HostAppListener.intercept()` — intercept and optionally consume JS bridge messages
- `MiniAppHeaderConfig` — configure top app bar (title, background color, foreground color, show/hide)
- `openMiniApp()` accepts optional `parameter: Map<String, List<String>>` appended as URL query params

### Changed
- `SdkConfig` now requires `partnerKey` and `secretId` (provided at AppCenter registration)
- `HostAppBridge` is now constructed via `HostAppBridgeImpl(hostApp: HostAppListener)`

## Requirements

- Gradle 8.0+
- Target Android SDK 24–34

## Distribution

```kotlin
implementation("vn.viettelpay:miniappsdk:2.0.20")
```
