# Release Notes — VDOMiniApp Android 2.0.20

**Version:** 2.0.20
**Platform:** Android

## Changelog

### Added

- `openMiniAppFromLinkQR()` — mở mini app từ Adjust link, Adjust QR, hoặc QR data link
- `openMiniAppActivity()` — mở mini app trong một Activity mới
- `SdkConfig.partnerKey` và `SdkConfig.secretId` — thông tin xác thực partner, cấp khi đăng ký
- `HostAppBridge.observeLifecycle()` — theo dõi thay đổi lifecycle của mini app
- `HostAppBridge.paymentStart()`, `paymentEnd()`, `paymentRequest()` — hooks cho luồng thanh toán
- `HostAppBridge.intercept()` — intercept JS bridge message từ mini app
- `MiniAppThemeConfig` — cấu hình toolbar (title, màu nền, màu chữ, ẩn/hiện)
- `openMiniApp()` nhận thêm tham số `parameter: Map<String, List<String>>` để truyền query params

### Changed

- `SdkConfig` nay yêu cầu `partnerKey` và `secretId`

## Requirements

- Gradle 8.0+
- minSdk 24
- compileSdk 34 / targetSdk 34

## Distribution

```kotlin
implementation("vn.viettelpay:miniappsdk:2.0.20")
```
