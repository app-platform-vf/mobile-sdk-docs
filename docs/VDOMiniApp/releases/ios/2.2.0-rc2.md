# Release Notes - iOS SDK 2.2.0-rc2

**Version:** 2.2.0-rc2
**Release date:** TBD

## Changelog

### Added
- `VDOMiniAppThemeConfig` — new unified theme configuration for Mini App header/toolbar (replaces `VDONavigationBarConfig`); includes `MiniAppLeftButton`, `MiniAppToolbarMode`, `MiniAppActionButtonThemeType`, and `MiniAppStatusBarForeground` enums
- `MiniAppLocation` — WGS-84 geographic coordinate struct for host app to supply location data (`latitude`, `longitude` as `Double?`)
- Toast notification system (`VDSToast`) for in-app messages
- Permission management UI: full-screen `PermissionManagementViewController` and `RemoveAllPermissionsBottomSheet`
- `MiniAppHeader` and `MiniAppMoreOptionBottomSheet` header/toolbar views
- JS bridge event `MINI_APP_TOKEN` — retrieve current Mini App login token
- JS bridge event `UPDATE_MINI_APP_THEME` — update full Mini App theme/header at runtime
- JS bridge event `SET_SWIPE_GESTURE_STATE` — unified control for swipe gestures

### Changed
- `getMiniDeepLinkUrlParameters()` return type changed from `[String: String]` to `[String: AnyCodableValue]`
- JS bridge theme events consolidated: `SET_BACKGROUND_STATUS_BAR_COLOR`, `UPDATE_STATUS_BAR_APPEARANCE`, `UPDATE_NAVIGATION_BAR_APPEARANCE`, `SET_NAVIGATION_BAR_COLOR` replaced by `UPDATE_MINI_APP_THEME`
- JS bridge swipe events unified: `SET_SWIPE_GESTURE_LEFT_STATE` and `SET_SWIPE_GESTURE_RIGHT_STATE` replaced by `SET_SWIPE_GESTURE_STATE`

### Removed
- `VDONavigationBarConfig` — replaced by `VDOMiniAppThemeConfig`; typealias `MiniAppThemeConfig = VDOMiniAppThemeConfig` provided for source-level compatibility
