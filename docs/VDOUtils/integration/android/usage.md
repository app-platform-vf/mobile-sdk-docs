# Android Usage

## Sử dụng VDOUtils

VDOUtils cung cấp các utilities dùng chung. Thông thường không cần gọi trực tiếp — Network SDK và MiniApp SDK sẽ sử dụng VDOUtils nội bộ.

## Tính năng chính

| Tính năng        | Mô tả                                        |
| ---------------- | --------------------------------------------- |
| Extensions       | Kotlin extensions cho standard types          |
| Helpers          | Utility functions dùng chung                  |
| Data Formatting  | Format date, number, string                   |
| Logging          | Logging utilities cho debug và monitoring     |
