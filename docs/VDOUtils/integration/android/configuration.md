# Android Configuration

## Khởi tạo SDK

VDOUtils không cần khởi tạo riêng — nó cung cấp utilities được sử dụng nội bộ bởi các SDK khác.

```kotlin
// VDOUtils được sử dụng nội bộ bởi Network SDK và MiniApp SDK
// Nếu dùng standalone, import trực tiếp
```

## Capabilities

- Utility extensions
- Data formatting helpers
- Logging level configuration
