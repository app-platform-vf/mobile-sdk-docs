# iOS Integration Overview

## Giới thiệu

VDOUtils SDK cung cấp các utilities dùng chung cho hệ SDK trên nền tảng iOS.

## Vai trò

| SDK          | Mô tả                                |
| ------------ | ------------------------------------- |
| `VDOUtils`   | Utilities dùng chung (SDK này)        |

> VDOUtils là base SDK — không có internal dependency. VDONetwork và VDOMiniApp phụ thuộc vào VDOUtils.
