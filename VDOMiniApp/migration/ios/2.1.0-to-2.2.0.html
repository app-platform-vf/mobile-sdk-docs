Migration Guide: 2.1.0 → 2.2.0

Breaking Changes​

1. VDONavigationBarConfig đã bị xóa​

VDONavigationBarConfig đã được thay thế hoàn toàn bởi VDOMiniAppThemeConfig.

Một typealias MiniAppThemeConfig = VDOMiniAppThemeConfig được cung cấp để giảm thiểu thay đổi tên type, nhưng các property của struct đã thay đổi hoàn toàn — cần cập nhật code khởi tạo.

Trước (2.1.0):

let navConfig = VDONavigationBarConfig(
    enable: true,
    title: "My MiniApp",
    tintColor: .white,
    isShowTitle: true,
    backgroundColor: .blue,
    isShowButtonBack: true
)

Sau (2.2.0):

let theme = VDOMiniAppThemeConfig(
    headerColor: "#0000FF",
    headerTitle: "My MiniApp",
    textColor: "#FFFFFF",
    leftButton: .back,
    toolbarMode: .normal
)

2. getMiniDeepLinkUrlParameters() thay đổi return type​

Return type thay đổi từ [String: String] sang [String: AnyCodableValue].

Trước (2.1.0):

let params: [String: String] = miniApp.getMiniDeepLinkUrlParameters()
let value: String = params["key"] ?? ""

Sau (2.2.0):

let params: [String: AnyCodableValue] = miniApp.getMiniDeepLinkUrlParameters()
let value = "\(params["key"] ?? "")"

3. JS Bridge events thay đổi​

Các event điều khiển giao diện header cũ đã được gộp thành một event duy nhất.

Event cũ (2.1.0)Event mới (2.2.0)
SET_BACKGROUND_STATUS_BAR_COLORUPDATE_MINI_APP_THEME
UPDATE_STATUS_BAR_APPEARANCEUPDATE_MINI_APP_THEME
UPDATE_NAVIGATION_BAR_APPEARANCEUPDATE_MINI_APP_THEME
SET_NAVIGATION_BAR_COLORUPDATE_MINI_APP_THEME

Mini App JS cần cập nhật để dùng UPDATE_MINI_APP_THEME thay thế 4 event trên.

4. JS Bridge swipe gesture events thay đổi​

Event cũ (2.1.0)Event mới (2.2.0)
SET_SWIPE_GESTURE_LEFT_STATESET_SWIPE_GESTURE_STATE
SET_SWIPE_GESTURE_RIGHT_STATESET_SWIPE_GESTURE_STATE

SET_SWIPE_GESTURE_STATE kiểm soát tất cả swipe gesture (trái và phải) bằng một tham số duy nhất.
